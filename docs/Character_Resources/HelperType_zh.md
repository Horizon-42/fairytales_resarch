推荐 Tag (Code),英文分类名称,权威出处 (Thompson Motif Range),典型角色与功能
1. ANIMAL,The Helpful Animal,B300 - B599,动物。报恩的老鼠、载人飞鹰、说话的马。利用生物特性帮助主角。
2. SUPERNATURAL,Supernatural Being,"N810, F200-F399",超自然存在。仙女教母(Fairy)、灯神(Genie)。拥有固有法力。
3. COMPANION,Extraordinary Companion,F600 - F699,异能同伴。千里眼、顺风耳、大力士。通常成组出现 (ATU 513)。
4. DEAD,The Grateful Dead,E341 (ATU 505),感恩的死者。因主角安葬尸体而回魂报恩的鬼魂或灵体。
5. MAIDEN,The Helper Maiden,"N831, G530.2, T170",被囚/反派之女。阿里阿德涅(线团)、美狄亚。既是被救者又是智囊/施法者。
6. HUMAN,The Human Helper,N820 - N850,普通人类。森林里的老妇人(指路人)、乞丐。通常提供信息而非魔法。
7. OBJECT,Magic Object (Active),"D800+, D1600+",自主物品。会说话的镜子、自己打人的棍子。必须具有自主意识。
