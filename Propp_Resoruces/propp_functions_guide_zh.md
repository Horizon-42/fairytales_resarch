# Vladimir Propp's Morphology of the Folktale: A Structural Guide
# 普罗普民间故事形态学：功能索引与标注指南

**Source / 出处**: Vladimir Propp, *Morphology of the Folktale* (1928/1968).
**Core Concept / 核心概念**: Functions are stable, constant elements in a tale, independent of who performs them.
**Note**: Do not interpret functions literally; focus on their **structural role** in the narrative flow.

---

## Part 1: Bilingual Detailed Index (中英对照详注版)

### Phase 1: The Preparatory Section (准备阶段)
*Setting the stage for conflict. / 为冲突搭建舞台。*

| Symbol | Function (En) | 功能 (Cn) | **Structural Meaning (结构内涵与深层逻辑)** |
| :--- | :--- | :--- | :--- |
| **$\alpha$** | **Initial Situation** | **初始情境** | 介绍家庭成员、英雄身份。这是平静被打破前的状态。 |
| **$\beta$** | **Absentation** | **缺席** | 权威长辈离开（如父母外出），导致保护机制失效，**秩序出现真空**。 |
| **$\gamma$** | **Interdiction** | **禁令** | 英雄被警告不要做某事（“别开门”、“别去森林”）。这是**设立边界**。 |
| **$\delta$** | **Violation** | **违反** | 英雄打破禁令。这是**坏事发生的必要条件**。反派通常在此刻悄然入场。 |
| **$\epsilon$** | **Reconnaissance** | **侦察** | 反派搜集情报（“谁住在这？”）。反派寻找受害者的弱点。 |
| **$\zeta$** | **Delivery** | **获悉** | 反派获得了情报。防御体系出现漏洞。 |
| **$\eta$** | **Trickery** | **欺骗** | 反派伪装自己（如狼外婆）试图欺骗受害者。 |
| **$\theta$** | **Complicity** | **同谋/上钩** | 受害者被骗并协助了敌人（如给坏人开门）。**防御彻底瓦解**。 |

### Phase 2: The Complication (纠纷与开端)
*The story properly begins. / 英雄使命的确立。*

| Symbol | Function (En) | 功能 (Cn) | **Structural Meaning (结构内涵与深层逻辑)** |
| :--- | :--- | :--- | :--- |
| **A** | **Villainy** | **加害** | 反派造成实质伤害（绑架、谋杀、盗窃）。**故事的核心驱动力 (Type 1)**。 |
| **a** | **Lack** | **匮乏** | (与 A 互斥) 无反派作恶，但主角**缺少**某物（求子、求药）。**故事的核心驱动力 (Type 2)**。 |
| **B** | **Mediation** | **调解/知晓** | 坏消息传开，英雄被请求或被派遣去解决问题。 |
| **C** | **Counteraction** | **决定反抗** | 英雄决定出发（Seeker Hero 的意志体现）。 |
| **$\uparrow$** | **Departure** | **离家/出发** | 英雄离开家。物理空间的转移，标志着**进入冒险世界**。 |

### Phase 3: The Donor Sequence (赠与者序列)
*Acquiring the means to win. / 获取魔法与助手。*

| Symbol | Function (En) | 功能 (Cn) | **Structural Meaning (结构内涵与深层逻辑)** |
| :--- | :--- | :--- | :--- |
| **D** | **1st Function of Donor** | **赠与者的考验** | **关键点**：这通常是**测试**而非直接赠送。英雄遭遇乞丐、动物或盘问。 |
| **E** | **Hero's Reaction** | **英雄的反应** | 英雄对测试做出回应（分享食物、回答问题、通过考验）。 |
| **F** | **Receipt of Agent** | **获得魔法剂** | 因反应正确，英雄获得宝物或助手 (Helper)。**能力的获取**。 |

### Phase 4: The Struggle (斗争)
*Direct confrontation. / 直接对抗。*

| Symbol | Function (En) | 功能 (Cn) | **Structural Meaning (结构内涵与深层逻辑)** |
| :--- | :--- | :--- | :--- |
| **G** | **Guidance** | **指引/转移** | 英雄被带到反派或宝藏所在地（如骑狼、飞毯）。**空间跨越**。 |
| **H** | **Struggle** | **交锋/斗争** | 英雄与反派直接对抗（战斗、比赛、打牌）。 |
| **J** | **Branding** | **标记** | 英雄受伤或获得身体印记（将在结尾 Q 用于识别）。 |
| **I** | **Victory** | **战胜** | 反派被击败（死亡、赶走、认输）。 |
| **K** | **Liquidation** | **消除灾祸** | 初始的 A (加害) 或 a (匮乏) 被解决。人被救出，宝物到手。 |

### Phase 5: The Return & Recognition (归来与结局)
*The final test of status. / 身份确认与最终奖赏。*

| Symbol | Function (En) | 功能 (Cn) | **Structural Meaning (结构内涵与深层逻辑)** |
| :--- | :--- | :--- | :--- |
| **$\downarrow$** | **Return** | **归来** | 英雄踏上归途。 |
| **Pr** | **Pursuit** | **追捕** | 英雄在归途中被追杀（通常是 Obstacle Flight 发生处）。 |
| **Rs** | **Rescue** | **救援** | 英雄获救（如躲进树洞、扔障碍物阻挡）。 |
| **o** | **Unrecognized Arrival**| **隐身到达** | 英雄回到家乡，但未被认出（或故意伪装）。 |
| **L** | **Unfounded Claims** | **无端要求** | **假英雄**（False Hero，如哥哥/大臣）冒领功劳。 |
| **M** | **Difficult Task** | **难题** | 为了辨别真假，英雄面临极难的任务（举重石、喝滚油）。 |
| **N** | **Solution** | **完成** | 英雄解决了难题。 |
| **Q** | **Recognition** | **认出** | 通过难题或标记 (J)，大家认出了真英雄。 |
| **Ex** | **Exposure** | **败露** | 假英雄或反派的伪装被揭穿。 |
| **T** | **Transfiguration** | **变形/焕然一新** | 英雄获得新外貌（变帅、金身、穿龙袍）。**本质的升华**。 |
| **U** | **Punishment** | **惩罚** | 反派或假英雄被处决。 |
| **W** | **Wedding** | **婚礼/结局** | **终极奖赏**。结婚、登基、成仙或发财。**社会秩序的重建与提升**。 |

## Part 3: Annotation Guidelines & Expert Tips (标注建议)

在训练 LLM 或进行数据标注时，请遵循以下核心逻辑，而非字面意思。

### 1. The "Donor Triangle" (D-E-F 铁三角)
不要孤立地标注“F (获得宝物)”。F 的存在通常依赖于前置的测试。
* **错误逻辑**: 主角捡到一把剑 -> F。
* **正确逻辑**: 主角遇到老人 (D) -> 主角给老人水喝 (E) -> 老人送主角剑 (F)。
* **应用**: 如果只有“相遇”而无后续获得助力的结果，通常只是连接性情节；如果有“结拜/同行”，则该相遇也是 Donor 序列的一部分。

### 2. A vs. a (故事的发动机)
每个故事通常只有一个核心驱动力，必须在开头明确区分。
* **Type A (Villainy)**: 主要是**恢复**（把被抓的人找回来）。
* **Type a (Lack)**: 主要是**获取**（把没有的东西找回来）。

### 3. W (Wedding) implies "Ultimate Reward"
不要被“Wedding”这个词限制。
* **场景**: 哪吒肉身成圣，被封为神将。
* **标注**: **W** (Wedding/Accession to the throne)。
* **理由**: 这是社会地位的最终确认和奖赏。

### 4. Handling "Obstacle Flight" (神奇逃亡)
这是 **Pr (Pursuit)** 和 **Rs (Rescue)** 的组合拳。
* **Pr**: 反派追击。
* **Rs**: 主角扔出梳子变森林（Rs），阻挡了反派。

### 5. "Encounter" is not a Function
“相遇”本身不是功能，要看相遇带来了什么。
* 遇到反派 -> **$\epsilon$** (侦察) 或 **H** (斗争)。
* 遇到同伴/老人 -> **D** (考验) 或 **F** (获得助手)。
* 遇到求救者 -> **B** (调解/任务发布)。
